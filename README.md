# 🚀 KaamOnClick
### One Tap. Problem Gone.

**KaamOnClick** is a hyperlocal service marketplace platform that connects customers with nearby service professionals such as plumbers, electricians, tutors, and other home service providers.

Instead of searching for service providers individually, users can **post a job request and receive bids from professionals**, then select the best provider based on price, experience, and service details.

The platform demonstrates a **real-world marketplace architecture** with authentication, job posting, bidding, and location-based service discovery.

---

## ✨ Core Features

🔐 **Authentication System**
- Secure signup and login
- JWT-based authentication
- Role-based users

📢 **Job Posting**
- Customers can create service requests
- Specify service type, location, and job details

💰 **Bidding System**
- Service providers can view jobs in their state
- Providers submit price bids
- Customers choose the best provider

📍 **Location Based Matching**
- Providers only see jobs from their registered state
- Enables hyperlocal service discovery
